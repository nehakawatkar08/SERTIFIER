# 📖 Overview

**SERTIFIER** is an Enterprise Digital Credential Management System developed to simplify the creation, management, and verification of digital certificates and identification cards.

The platform enables organizations, educational institutes, training centers, and enterprises to generate secure digital credentials with QR-code-based verification, ensuring authenticity and preventing document forgery.

Instead of manually designing and issuing certificates, administrators can generate personalized credentials instantly using predefined templates and centralized data management.

# 🚀 Installation

## 1 Clone Repository

```bash
git clone https://github.com/nehakawatkar08/SERTIFIER.git
```

---

## 2 Navigate to Project

```bash
cd SERTIFIER
```

---

## 3 Create Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv

source venv/bin/activate
```

---

## 4 Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 5 Apply Migrations

```bash
python manage.py migrate
```

---

## 6 Run Development Server

```bash
python manage.py runserver
```

Open your browser:

```
http://127.0.0.1:8000/
```
